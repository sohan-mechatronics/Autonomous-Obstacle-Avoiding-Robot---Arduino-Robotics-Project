# Autonomous Obstacle Avoiding Robot - Arduino Robotics Project
🤖 Arduino-Based Autonomous Obstacle Avoiding Robot  Designed and developed an Arduino-based autonomous robot capable of detecting and avoiding obstacles in real time using an HC-SR04 ultrasonic sensor and motor driver module.  🔧 Technologies Used: • Arduino UNO • HC-SR04 Ultrasonic Sensor • Motor Driver Module • DC Motors  🎯 Key Features: • Real-time obstacle detection • Automatic direction control • Collision avoidance • Autonomous navigation  This project demonstrates the practical application of robotics, embedded systems, and automation concepts.
